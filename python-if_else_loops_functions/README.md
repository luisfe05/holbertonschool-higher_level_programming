# Python - if/else, loops, functions

Introduction to conditionals, loops, and functions in Python 3.

## Files

| File | Description |
|------|-------------|
| `0-positive_or_negative.py` | Prints whether a random number is positive, negative, or zero |
| `1-last_digit.py` | Prints the last digit of a random number with conditions |
| `2-print_alphabet.py` | Prints the lowercase alphabet in one line using a loop |
| `3-print_alphabt.py` | Prints the lowercase alphabet excluding the letters `e` and `q` |
| `4-print_hexa.py` | Prints numbers from 0 to 98 in decimal and hexadecimal |
| `5-print_comb2.py` | Prints numbers from 00 to 99 separated by commas |
| `6-print_comb3.py` | Prints all possible combinations of two different digits |
| `7-islower.py` | Function that checks if a character is lowercase using ASCII |
| `8-uppercase.py` | Function that prints a string in uppercase using ASCII |
| `9-print_last_digit.py` | Function that prints and returns the last digit of a number |
| `10-add.py` | Function that adds two integers and returns the result |
| `11-pow.py` | Function that computes a to the power of b and returns the result |
| `12-fizzbuzz.py` | Function that prints FizzBuzz from 1 to 100 |
