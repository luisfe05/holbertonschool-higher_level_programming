# Python - Hello, World

Introduction to Python scripting. This project covers basic output, string manipulation, and formatting using Python 3.

## Files

| File | Description |
|------|-------------|
| `2-print.py` | Prints a string containing a double quote using the `print` function |
| `3-print_number.py` | Prints an integer stored in a variable using f-strings |
| `4-print_float.py` | Prints a float stored in a variable with 2 decimal places using f-strings |
| `5-print_string.py` | Prints a string 3 times, then prints its first 9 characters |
| `6-concat.py` | Concatenates two string variables and prints the result |
| `7-edges.py` | Uses string slicing to extract the first 3 letters, last 2 letters, and middle characters of a word |
| `8-concat_edges.py` | Builds a new sentence by slicing an existing string variable |
| `9-easter_egg.py` | Prints The Zen of Python using the `import this` easter egg |
