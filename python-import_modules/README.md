# Python - import & modules

Introduction to importing functions, variables, and modules in Python 3.

## Files

| File | Description |
|------|-------------|
| `0-add.py` | Imports the `add` function and prints the result of `1 + 2 = 3` |
| `1-calculation.py` | Imports calculator functions and prints results of basic math operations |
| `2-args.py` | Prints the number and list of command line arguments |
| `3-infinite_add.py` | Prints the sum of all command line arguments |
| `4-hidden_discovery.py` | Prints all names defined in a compiled module that do not start with `__` |
| `5-variable_load.py` | Imports a variable from a file and prints its value |
