# Holberton School Higher Level Programming - Python Data Structures

This directory contains tasks and solutions covering fundamentals of core Python data structures, focusing on efficiency, clean layout syntax, and manipulation techniques without relying on external dependencies or modules.

---

## Project Directories & Chapters

### 1. Python - Data Structures: Lists, Tuples
* **Directory:** `python-data_structures`

| Task Name | Source File | Description |
| :--- | :--- | :--- |
| 0. Print a list of integers | `0-print_list_integer.py` | Prints all integers of a list, one per line. |
| 1. Secure access to an element | `1-element_at.py` | Retrieves an element from a list like in C. |
| 2. Replace element | `2-replace_in_list.py` | Replaces an element of a list at a specific position. |
| 3. Print a list of integers... in reverse! | `3-print_reversed_list_integer.py` | Prints all integers of a list in reverse order. |
| 4. Replace in a copy | `4-new_in_list.py` | Replaces an element in a list without modifying the original. |
| 5. Can you dive into list? | `5-no_c.py` | Removes all characters `c` and `C` from a string. |
| 6. Lists of lists = Matrix | `6-print_matrix_integer.py` | Prints a matrix of integers using `str.format()`. |
| 7. Tuples addition | `7-add_tuple.py` | Adds 2 tuples containing 2 integers each. |
| 8. More returns! | `8-multiple_returns.py` | Returns a tuple with the length of a string and its first character. |
| 9. Find the max | `9-max_integer.py` | Finds the biggest integer of a list without using built-ins. |
| 10. Only by 2 | `10-divisible_by_2.py` | Finds all multiples of 2 in a list and maps them to a Boolean list. |
| 11. Delete at | `11-delete_at.py` | Deletes an item at a specific position in a list without using `pop()`. |
| 12. Switch | `12-switch.py` | Swaps the values of variables `a` and `b` in a 5-line script. |
